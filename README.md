### Obsidian

以关联知识图谱的方式查看：

- 下载并安装 [Obsidian](https://obsidian.md/)
- 下载代码库 `git clone git@github.com:zhugelianglongming/puzzle.git`
- 点击`打开本地仓库`，选择代码库本地根目录
- 点击`查看关系图谱`
