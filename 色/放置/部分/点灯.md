# 规则

## 解答方式

- 将**部分**白格标记为`灯`

## 限制

| 序号  | 限制区域 | 限制规则                   | 备注      |
|:---:|:----:|:-----------------------|:--------|
|  1  |  黑格  | 阻挡视线                   |         |
|  2  |  白格  | 所有白格至少可以看到 1 盏灯        | 水平或垂直方向 |
|  3  |  灯   | 所有灯看不到其他灯              | 水平或垂直方向 |
|  3  | 提示数  | 提示数 `X`：标记格的共边邻格有`X`盏灯 |         |
